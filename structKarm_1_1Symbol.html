Detailed Description

A symbol is a unique string that is interned in a global registry. It is used to represent identifiers in a way that allows for fast comparisons and lookups. Symbols are immutable and can be compared by pointer equality. They are typically used for identifiers in languages, such as HTML tags, attributes, and other names that are used frequently and need to be compared often.
